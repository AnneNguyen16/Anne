# Number row characters
- !: exclamation mark, chấm than.
- @: At sign, a còng.
- \#: hashtag, number sign.
- $: dollar, currency, tiền tệ, đô la.
- %: percentage, phần trăm.
- ^: Circumflex.
- &: Ampersand, and, plus, và.
- \*: Asterisk, Dinkus, dấu sao. 
- (): Parenthesis, dấu ngoặc.
